2.7. Fedora¶

2.7.1. Fedora 21¶

Fedora 21ではMariaDB 10.0.xが採用されました。MroongaはMariaDB 10.0.15以降バンドルされるようになったので、MariaDBをインストールすることで使えます。

バンドルされているMroongaのバージョンはやや古いです。
